# Fegeya Translatfe
## Very simple language translator written in C++.

* For Educational purposes only.
* Supports simple x syntax.

* x
  * English-to-German, Turkish.
  * String-to-Morse, Morse-to-String.

#### Why?
  * For:
     * Learning new words, verbs..
     * Learn language grammar.
     * Fun!
